下面给你一版**实现草案版**，目标不是一次性可运行的完整工程，而是给团队一个**可以直接开工拆任务**的 TypeScript 代码骨架。它遵循前面那版技术设计：`MEMORY.md` 只做高价值总纲，`memory/YYYY-MM-DD.md` 记录情节记忆，`semantic/*` 和 `procedures/*` 承载长期可检索细节；检索层默认假设复用 OpenClaw 已有的记忆检索能力，而不是重造一套底层索引。[Source](https://docs.openclaw.ai/concepts/system-prompt) [Source](https://docs.openclaw.ai/concepts/memory) [Source](https://docs.openclaw.ai/concepts/memory-builtin)

同时，这版实现草案把“Dreaming/巩固”的核心思想具体化成可维护的工程模块：**扫描器、评分器、归档器、检索器、汇总器**。这样既兼容 OpenClaw 当前“Markdown 为事实源”的机制，也保留了后续接入更强 rerank、图谱层、外部插件的扩展空间。[Source](https://docs.openclaw.ai/concepts/dreaming) [Source](https://docs.openclaw.ai/concepts/memory-search)

# 《[OpenClaw](https://docs.openclaw.ai/concepts/memory) 记忆系统实现草案版》

## 1. 实现边界与假设

这套代码骨架假设你们的系统遵循以下边界：

第一，**Markdown 是事实主存**。长期事实、程序经验、daily notes 都在 workspace 文件系统里，前置解析层负责 frontmatter 与正文抽取。这个边界与 OpenClaw 官方“记忆就是落盘文件，没有隐藏状态”的定义一致。[Source](https://docs.openclaw.ai/concepts/memory)

第二，**检索索引不是这里重做**。Builtin memory engine 已经提供 per-agent SQLite、FTS5、向量检索与 hybrid search，因此这份代码里检索器采用 `SearchAdapter` 抽象，让你可以接 OpenClaw 原生 `memory_search` 能力或未来的自研索引层，而不是在这里从零实现 embedding pipeline。[Source](https://docs.openclaw.ai/concepts/memory-builtin) [Source](https://docs.openclaw.ai/concepts/memory-search)

第三，**`MEMORY.md` 自动汇总，不人工堆积**。因为官方明确说明它会进入每轮上下文，应尽量简洁，所以实现上把它视为“从长期层反向生成的摘要文件”，而不是人手维护的总仓库。[Source](https://docs.openclaw.ai/concepts/system-prompt)

## 5. 目录扫描规则

OpenClaw 的记忆核心仍然是文件，所以第一层一定是扫描器。扫描器的职责不是做“智能”，而是**可靠发现、分类、校验、解析**。由于 OpenClaw daily notes 与长期主题文件的角色不同，扫描器应把 daily notes 和 semantic/procedural 文档分开处理。[Source](https://docs.openclaw.ai/concepts/memory)

### 5.1 扫描规则定义

```ts

  requiresFrontmatter: boolean;
  parseMode: "markdown" | "daily-note";
}

const DAILY_NOTE_REGEX = /^memory\/\d{4}-\d{2}-\d{2}\.md$/;
const SEMANTIC_REGEX = /^memory\/semantic\/.+\.md$/;
const PROCEDURAL_REGEX = /^memory\/procedures\/.+\.md$/;
const REFLECTION_REGEX = /^memory\/reflection\/.+\.md$/;
const ARCHIVE_REGEX = /^memory\/archive\/.+\.md$/;

export const scanRules: ScanRule[] = [
  {
```

### 5.2 扫描器骨架

```ts
// src/memory/scanner/workspace-scanner.ts

import path from "node:path";
import fs from "node:fs/promises";
import { scanRules } from "./scan-rules";
import { MemoryDocument } from "../types/memory";
import { parseMarkdownDocument, parseDailyNote } from "../utils/frontmatter";

export class WorkspaceScanner {
  constructor(private workspaceRoot: string) {}

  async scanAll(): Promise<MemoryDocument[]> {
    const files = await this.listMarkdownFiles(this.workspaceRoot);
    const docs: MemoryDocument[] = [];

    for (const absPath of files) {
      const relPath = this.toWorkspaceRelative(absPath);
      const rule = scanRules.find(r => r.match(relPath));
      if (!rule) continue;

      const raw = await fs.readFile(absPath, "utf-8");
      const doc =
        rule.parseMode === "daily-note"
          ? parseDailyNote(relPath, raw)
          : parseMarkdownDocument(relPath, raw, rule.classify(relPath), rule.requiresFrontmatter);

      docs.push(doc);
    }

    return docs;
  }

  async scanByKind(kind: ReturnType<(typeof scanRules)[number]["classify"]>): Promise<MemoryDocument[]> {
    const all = await this.scanAll();
    return all.filter(doc => doc.layer === kind || (kind === "archive" && doc.path.startsWith("memory/archive/")));
  }

  private async listMarkdownFiles(root: string): Promise<string[]> {
    const output: string[] = [];

    const walk = async (dir: string) => {
      const entries = await fs.readdir(dir, { withFileTypes: true });
      for (const entry of entries) {
        const abs = path.join(dir, entry.name);
        if (entry.isDirectory()) {
          await walk(abs);
        } else if (entry.isFile() && entry.name.endsWith(".md")) {
          output.push(abs);
        }
      }
    };

    await walk(root);
    return output;
  }

  private toWorkspaceRelative(absPath: string): string {
    return path.relative(this.workspaceRoot, absPath).replace(/\\/g, "/");
  }
}
```

### 5.3 Frontmatter 解析工具骨架

```ts
// src/memory/utils/frontmatter.ts

import matter from "gray-matter";
import { MemoryDocument, MemoryFrontmatter, DailyNoteEvent } from "../types/memory";

export function parseMarkdownDocument(
  relPath: string,
  raw: string,
  layer: MemoryDocument["layer"],
  requiresFrontmatter: boolean
): MemoryDocument {
  const parsed = matter(raw);

  if (requiresFrontmatter && (!parsed.data || Object.keys(parsed.data).length === 0)) {
    throw new Error(`frontmatter required for ${relPath}`);
  }

  const fm = normalizeFrontmatter(parsed.data as Partial<MemoryFrontmatter>);

  return {
    path: relPath,
    layer,
    frontmatter: fm,
    title: fm?.title,
    summary: extractSummary(parsed.content),
    content: parsed.content.trim(),
    raw
  };
}

export function parseDailyNote(relPath: string, raw: string): MemoryDocument {
  return {
    path: relPath,
    layer: "episodic",
    content: raw,
    raw,
    summary: extractSummary(raw)
  };
}

function normalizeFrontmatter(input?: Partial<MemoryFrontmatter>): MemoryFrontmatter | undefined {
  if (!input) return undefined;

  return {
    id: input.id ?? "",
    type: input.type!,
    title: input.title ?? "",
    status: input.status ?? "active",
    confidence: input.confidence ?? 0.7,
    priority: input.priority ?? "medium",
    scope: input.scope ?? "project",
    owner: input.owner ?? "agent",
    createdAt: input.createdAt ?? new Date().toISOString(),
    updatedAt: input.updatedAt ?? new Date().toISOString(),
    lastVerifiedAt: input.lastVerifiedAt,
    lastUsedAt: input.lastUsedAt,
    reviewAfter: input.reviewAfter,
    sourceRefs: input.sourceRefs ?? [],
    tags: input.tags ?? [],
    aliases: input.aliases ?? [],
    relatedIds: input.relatedIds ?? [],
    supersedes: input.supersedes ?? [],
    supersededBy: input.supersededBy ?? [],
    sensitivity: input.sensitivity ?? "normal",
    retention: input.retention ?? "standard",
    archivePolicy: input.archivePolicy ?? "auto",
    validation: input.validation
  };
}

function extractSummary(content: string): string {
  return content.split("\n").find(Boolean)?.slice(0, 200) ?? "";
}

export function parseDailyEvents(raw: string, file: string): DailyNoteEvent[] {
  const lines = raw.split("\n");
  const events: DailyNoteEvent[] = [];
  let currentTime = "00:00";
  let currentKind: DailyNoteEvent["kind"] = "other";

  for (const line of lines) {
    const headingMatch = line.match(/^##\s+(\d{2}:\d{2}).*?(User Request|Decision|Result|Incident|Insight)?/i);
    if (headingMatch) {
      currentTime = headingMatch[1];
      const tag = (headingMatch[2] || "").toLowerCase();
      currentKind =
        tag.includes("request") ? "request" :
        tag.includes("decision") ? "decision" :
        tag.includes("result") ? "result" :
        tag.includes("incident") ? "incident" :
        tag.includes("insight") ? "insight" : "other";
      continue;
    }

    if (line.trim().startsWith("- ")) {
      events.push({
        timestamp: currentTime,
        kind: currentKind,
        text: line.replace(/^- /, "").trim(),
        sourceRef: { file, anchor: currentTime }
      });
    }
  }

  return events;
}
```

## 6. 记忆评分器代码骨架

官方 Dreaming 的 Deep ranking 依赖频率、相关性、查询多样性、时效性、多日巩固等信号，所以这里把评分器做成显式模块，便于以后调参与 A/B 测试。[Source](https://docs.openclaw.ai/concepts/dreaming)

```ts
// src/memory/scoring/memory-scorer.ts

import { MemoryCandidate } from "../types/memory";
import { ScoredCandidate, ScoreBreakdown, ScoringContext } from "../types/scoring";
import { MemoryConfig } from "../config/memory-config";

export class MemoryScorer {
  constructor(private config: MemoryConfig) {}

  score(candidate: MemoryCandidate, ctx: ScoringContext): ScoredCandidate {
    const frequency = clamp01(normalizeCount(ctx.recallCount, 5));
    const relevance = clamp01(ctx.avgRecallQuality);
    const queryDiversity = clamp01(normalizeCount(ctx.uniqueQueries, 4));
    const recency = clamp01(scoreRecency(ctx.lastSeenAt));
    const consolidation = clamp01(normalizeCount(ctx.daysObserved, 3));
    const structureValue = clamp01(scoreStructureValue(candidate));
    const userConfirmation = ctx.userConfirmed ? 1 : 0;
    const validation = clamp01(scoreValidation(ctx.validationSuccessCount, ctx.validationFailureCount));

    const penalties = this.computePenalties(ctx);
    const finalScore = clamp01(this.computeBaseScore(candidate, {
      frequency,
      relevance,
      queryDiversity,
      recency,
      consolidation,
      structureValue,
      userConfirmation,
      validation
    }) - penalties);

    const threshold = this.resolveThreshold(candidate);
    const breakdown: ScoreBreakdown = {
      frequency,
      relevance,
      queryDiversity,
      recency,
      consolidation,
      structureValue,
      userConfirmation,
      validation,
      penalties,
      finalScore
    };

    return {
      candidate,
      breakdown,
      threshold,
      promotionEligible: finalScore >= threshold
    };
  }

  private computeBaseScore(
    candidate: MemoryCandidate,
    s: Omit<ScoreBreakdown, "penalties" | "finalScore">
  ): number {
    if (candidate.layer === "semantic") {
      return (
        0.20 * s.frequency +
        0.20 * s.relevance +
        0.15 * s.queryDiversity +
        0.10 * s.recency +
        0.15 * s.consolidation +
        0.10 * s.structureValue +
        0.10 * s.userConfirmation
      );
    }

    return (
      0.15 * s.frequency +
      0.15 * s.relevance +
      0.10 * s.queryDiversity +
      0.10 * s.recency +
      0.15 * s.consolidation +
      0.10 * s.structureValue +
      0.05 * s.userConfirmation +
      0.20 * s.validation
    );
  }

  private resolveThreshold(candidate: MemoryCandidate): number {
    if (candidate.proposedType === "semantic.user") {
      return this.config.thresholds.semanticUser;
    }
    if (candidate.proposedType === "semantic.project") {
      return this.config.thresholds.semanticProject;
    }
    if (candidate.proposedType === "procedural.runbook") {
      return this.config.thresholds.proceduralRunbook;
    }
    return candidate.layer === "semantic"
      ? this.config.thresholds.semantic
      : this.config.thresholds.procedural;
  }

  private computePenalties(ctx: ScoringContext): number {
    let score = 0;
    if (ctx.conflictDetected) score += 0.20;
    if (ctx.sensitivityDetected) score += 1.0; // 直接禁止晋升
    if (ctx.recallCount <= 1) score += 0.10;
    if ((ctx.validationFailureCount ?? 0) >= 2) score += 0.10;
    return score;
  }
}

function normalizeCount(value: number, cap: number): number {
  return Math.min(value / cap, 1);
}

function scoreRecency(lastSeenAt?: string): number {
  if (!lastSeenAt) return 0.3;
  const days = diffDays(new Date(lastSeenAt), new Date());
  if (days <= 3) return 1;
  if (days <= 7) return 0.8;
  if (days <= 30) return 0.6;
  if (days <= 90) return 0.3;
  return 0.1;
}

function scoreStructureValue(candidate: MemoryCandidate): number {
  const factDensity = Math.min(candidate.facts.length / 5, 1);
  const hasRefs = candidate.sourceRefs.length > 0 ? 0.2 : 0;
  return clamp01(factDensity * 0.8 + hasRefs);
}

function scoreValidation(success = 0, failure = 0): number {
  const total = success + failure;
  if (total === 0) return 0;
  return clamp01(success / total);
}

function clamp01(value: number): number {
  return Math.max(0, Math.min(1, value));
}

function diffDays(a: Date, b: Date): number {
  return Math.floor(Math.abs(b.getTime() - a.getTime()) / (1000 * 60 * 60 * 24));
}
```

## 7. 归档器代码骨架

归档器的目标不是删除一切旧东西，而是把“过时但值得保留审计轨迹”的内容下沉到 archive 层。这和长期记忆治理要求一致，也避免活跃层越来越脏。[Source](https://docs.openclaw.ai/concepts/memory)

```ts
// src/memory/archive/memory-archiver.ts

import fs from "node:fs/promises";
import path from "node:path";
import { MemoryConfig } from "../config/memory-config";
import { ArchiveDecision, ArchiveReport, ArchiveReportItem } from "../types/archive";
import { MemoryDocument } from "../types/memory";

export class MemoryArchiver {
  constructor(private config: MemoryConfig) {}

  decide(doc: MemoryDocument): ArchiveDecision | null {
    const fm = doc.frontmatter;
    if (!fm) return null;
    if (fm.archivePolicy === "never") return null;
    if (fm.status === "archived") return null;

    const now = new Date();

    if (doc.layer === "semantic") {
      if (fm.supersededBy && fm.supersededBy.length > 0) {
        return { doc, mode: "archive", reason: "superseded" };
      }

      if (isOlderThan(fm.lastUsedAt ?? fm.updatedAt, this.config.retention.semanticArchiveAfterDays, now)) {
        return { doc, mode: "archive", reason: "stale_semantic" };
      }

      if (fm.confidence < 0.45) {
        return { doc, mode: "demote", reason: "low_confidence" };
      }
    }

    if (doc.layer === "procedural") {
      const validation = fm.validation;
      if ((validation?.failureCount ?? 0) >= 2) {
        return { doc, mode: "demote", reason: "repeated_validation_failures" };
      }

      if (isOlderThan(
        validation?.lastValidatedAt ?? fm.lastVerifiedAt ?? fm.updatedAt,
        this.config.retention.proceduralArchiveAfterDays,
        now
      )) {
        return { doc, mode: "archive", reason: "stale_procedural" };
      }
    }

    if (doc.layer === "episodic") {
      const dateStr = extractDateFromDailyPath(doc.path);
      if (dateStr && isOlderThan(dateStr, this.config.retention.episodicArchiveAfterDays, now)) {
        return { doc, mode: "archive", reason: "episodic_expired" };
      }
    }

    return null;
  }

  async execute(decisions: ArchiveDecision[]): Promise<ArchiveReport> {
    const report: ArchiveReport = {
      runAt: new Date().toISOString(),
      archived: [],
      demoted: [],
      deleted: []
    };

    for (const decision of decisions) {
      const item = await this.apply(decision);
      if (decision.mode === "archive") report.archived.push(item);
      if (decision.mode === "demote") report.demoted.push(item);
      if (decision.mode === "delete") report.deleted.push(item);
    }

    return report;
  }

  private async apply(decision: ArchiveDecision): Promise<ArchiveReportItem> {
    const from = path.join(this.config.paths.workspaceRoot, decision.doc.path);

    if (decision.mode === "archive") {
      const toRel = this.toArchivePath(decision.doc.path);
      const toAbs = path.join(this.config.paths.workspaceRoot, toRel);
      await fs.mkdir(path.dirname(toAbs), { recursive: true });
      await fs.rename(from, toAbs);

      return {
        id: decision.doc.frontmatter?.id,
        from: decision.doc.path,
        to: toRel,
        mode: "archive",
        reason: decision.reason
      };
    }

    if (decision.mode === "demote") {
      // 可选：在原文件 frontmatter 里改为 review_needed / archived
      return {
        id: decision.doc.frontmatter?.id,
        from: decision.doc.path,
        mode: "demote",
        reason: decision.reason
      };
    }

    await fs.unlink(from);
    return {
      id: decision.doc.frontmatter?.id,
      from: decision.doc.path,
      mode: "delete",
      reason: decision.reason
    };
  }

  private toArchivePath(relPath: string): string {
    if (relPath.startsWith("memory/archive/")) return relPath;
    return `memory/archive/${relPath.replace(/^memory\//, "")}`;
  }
}

function isOlderThan(dateStr: string | undefined, days: number, now: Date): boolean {
  if (!dateStr) return false;
  const date = new Date(dateStr);
  return (now.getTime() - date.getTime()) / (1000 * 60 * 60 * 24) > days;
}

function extractDateFromDailyPath(relPath: string): string | null {
  const match = relPath.match(/memory\/(\d{4}-\d{2}-\d{2})\.md$/);
  return match?.[1] ?? null;
}
```

### 8.3 检索器主体

```ts
// src/memory/retrieval/memory-retriever.ts

import { SearchAdapter } from "./search-adapter";
import { RetrievalPlanner } from "./retrieval-planner";
import { RecallQuery, RecallResult, RankedRecallItem, SearchHit } from "../types/retrieval";
import { MemoryDocument } from "../types/memory";
import { MarkdownStore } from "../store/markdown-store";

export class MemoryRetriever {
  constructor(
    private searchAdapter: SearchAdapter,
    private planner: RetrievalPlanner,
    private store: MarkdownStore
  ) {}

  async recall(input: RecallQuery): Promise<RecallResult> {
    const plan = this.planner.build(input.intent);
    const hits: SearchHit[] = [];

    if (plan.includeSummary) {
      const summaryDoc = await this.store.getMemorySummary();
      if (summaryDoc) {
        hits.push({
          path: summaryDoc.path,
          title: summaryDoc.title ?? "MEMORY.md",
          chunkText: summaryDoc.content,
          layer: "summary",
          semanticScore: 0.7,
          keywordScore: 0.7,
          hybridScore: 0.7
        });
      }
    }

    if (plan.searchSemantic) {
      hits.push(...await this.searchAdapter.search({
        query: input.query,
        topK: input.topK ?? 5,
        vectorWeight: plan.vectorWeight,
        keywordWeight: plan.keywordWeight,
        exactMatchBoost: input.exactMatchBoost ?? plan.exactMatchBoost,
        recencyBias: input.recencyBias ?? plan.recencyBias,
        filters: { layer: ["semantic"], includeArchived: input.includeArchived }
      }));
    }

    if (plan.searchProcedural) {
      hits.push(...await this.searchAdapter.search({
        query: input.query,
        topK: input.topK ?? 5,
        vectorWeight: plan.vectorWeight,
        keywordWeight: plan.keywordWeight,
        exactMatchBoost: input.exactMatchBoost ?? plan.exactMatchBoost,
        recencyBias: input.recencyBias ?? plan.recencyBias,
        filters: { layer: ["procedural"], includeArchived: input.includeArchived }
      }));
    }

    if (plan.searchEpisodic) {
      hits.push(...await this.searchAdapter.search({
        query: input.query,
        topK: input.topK ?? 5,
        vectorWeight: plan.vectorWeight,
        keywordWeight: plan.keywordWeight,
        exactMatchBoost: input.exactMatchBoost ?? plan.exactMatchBoost,
        recencyBias: input.recencyBias ?? plan.recencyBias,
        filters: { layer: ["episodic"], includeArchived: input.includeArchived }
      }));
    }

    const ranked = await this.rerank(input, dedupeHits(hits));
    return { items: ranked.slice(0, input.topK ?? 5), plan };
  }

  private async rerank(input: RecallQuery, hits: SearchHit[]): Promise<RankedRecallItem[]> {
    const items: RankedRecallItem[] = [];

    for (const hit of hits) {
      const doc = await this.safeLoadDocument(hit.path);
      const intentMatch = scoreIntentMatch(input.intent, hit.layer, doc);
      const authority = scoreAuthority(doc);
      const recencyScore = scoreRecency(doc);
      const diversity = 0.8; // 占位，后续可做 MMR/主题多样性
      const hybrid = hit.hybridScore ?? average(hit.semanticScore, hit.keywordScore);

      const recallRank =
        0.35 * intentMatch +
        0.30 * hybrid +
        0.15 * recencyScore +
        0.10 * authority +
        0.10 * diversity;

      items.push({
        document: doc,
        hit,
        recallRank,
        intentMatch,
        authority,
        diversity,
        recencyScore
      });
    }

    return items.sort((a, b) => b.recallRank - a.recallRank);
  }

  private async safeLoadDocument(relPath: string): Promise<MemoryDocument | undefined> {
    try {
      return await this.store.getByPath(relPath);
    } catch {
      return undefined;
    }
  }
}

function dedupeHits(hits: SearchHit[]): SearchHit[] {
  const map = new Map<string, SearchHit>();
  for (const hit of hits) {
    const key = `${hit.path}:${hit.chunkText.slice(0, 80)}`;
    const prev = map.get(key);
    if (!prev || (hit.hybridScore ?? 0) > (prev.hybridScore ?? 0)) {
      map.set(key, hit);
    }
  }
  return [...map.values()];
}

function scoreIntentMatch(intent: RecallQuery["intent"], layer?: string, doc?: MemoryDocument): number {
  if (intent === "preference" && layer === "semantic") return 1;
  if (intent === "procedure" && layer === "procedural") return 1;
  if (intent === "recent_activity" && layer === "episodic") return 1;
  if (intent === "error_lookup" && (layer === "procedural" || layer === "episodic")) return 0.9;
  if (intent === "fact" && layer === "semantic") return 0.9;
  return 0.5;
}

function scoreAuthority(doc?: MemoryDocument): number {
  const fm = doc?.frontmatter;
  if (!fm) return 0.5;

  let score = fm.confidence;
  if (fm.status === "superseded") score -= 0.3;
  if (fm.status === "review_needed") score -= 0.15;
  if ((fm.validation?.successCount ?? 0) > 0) score += 0.1;

  return clamp01(score);
}

function scoreRecency(doc?: MemoryDocument): number {
  const ts = doc?.frontmatter?.lastUsedAt ?? doc?.frontmatter?.updatedAt;
  if (!ts) return 0.4;
  const days = Math.floor((Date.now() - new Date(ts).getTime()) / 86400000);
  if (days <= 3) return 1;
  if (days <= 14) return 0.8;
  if (days <= 60) return 0.5;
  return 0.2;
}

function average(a = 0, b = 0): number {
  return (a + b) / 2;
}

function clamp01(n: number): number {
  return Math.max(0, Math.min(1, n));
}
```

## 9. MarkdownStore 骨架

存储层建议简单明确，不在这里塞太多业务规则。

```ts
// src/memory/store/markdown-store.ts

import fs from "node:fs/promises";
import path from "node:path";
import matter from "gray-matter";
import { MemoryConfig } from "../config/memory-config";
import { MemoryCandidate, MemoryDocument, MemoryFrontmatter } from "../types/memory";
import { parseMarkdownDocument } from "../utils/frontmatter";

export class MarkdownStore {
  constructor(private config: MemoryConfig) {}

  async getMemorySummary(): Promise<MemoryDocument | undefined> {
    return this.getByPath(this.config.paths.memorySummaryFile);
  }

  async getByPath(relPath: string): Promise<MemoryDocument> {
    const abs = path.join(this.config.paths.workspaceRoot, relPath);
    const raw = await fs.readFile(abs, "utf-8");

    const layer = relPath === "MEMORY.md"
      ? "summary"
      : relPath.startsWith("memory/semantic/")
      ? "semantic"
      : relPath.startsWith("memory/procedures/")
      ? "procedural"
      : relPath.startsWith("memory/")
      ? "episodic"
      : "reflection";

    return parseMarkdownDocument(relPath, raw, layer as MemoryDocument["layer"], layer !== "summary" && layer !== "episodic");
  }

  async upsertFromCandidate(candidate: MemoryCandidate): Promise<string> {
    const relPath = this.resolveCandidatePath(candidate);
    const absPath = path.join(this.config.paths.workspaceRoot, relPath);

    await fs.mkdir(path.dirname(absPath), { recursive: true });

    const existing = await tryRead(absPath);
    const merged = mergeCandidateIntoMarkdown(existing, candidate);

    await fs.writeFile(absPath, merged, "utf-8");
    return relPath;
  }

  async writeSummary(markdown: string): Promise<void> {
    const abs = path.join(this.config.paths.workspaceRoot, this.config.paths.memorySummaryFile);
    await fs.writeFile(abs, markdown, "utf-8");
  }

  private resolveCandidatePath(candidate: MemoryCandidate): string {
    if (candidate.layer === "semantic") {
      if (candidate.proposedType === "semantic.project") {
        return `memory/semantic/projects/${candidate.id}.md`;
      }
      if (candidate.proposedType === "semantic.entity") {
        return `memory/semantic/entities/${candidate.id}.md`;
      }
      return `memory/semantic/${candidate.id}.md`;
    }

    if (candidate.proposedType === "procedural.runbook") {
      return `memory/procedures/runbooks/${candidate.id}.md`;
    }
    if (candidate.proposedType === "procedural.lesson") {
      return `memory/procedures/lessons/${candidate.id}.md`;
    }
    return `memory/procedures/playbooks/${candidate.id}.md`;
  }
}

async function tryRead(absPath: string): Promise<string | null> {
  try {
    return await fs.readFile(absPath, "utf-8");
  } catch {
    return null;
  }
}

function mergeCandidateIntoMarkdown(existingRaw: string | null, candidate: MemoryCandidate): string {
  const fm: Partial<MemoryFrontmatter> = existingRaw
    ? matter(existingRaw).data
    : {};

  const mergedFrontmatter: MemoryFrontmatter = {
    id: candidate.id,
    type: candidate.proposedType,
    title: candidate.title,
    status: "active",
    confidence: Math.max(fm.confidence ?? 0.7, candidate.confidenceHint ?? 0.75),
    scope: candidate.scope,
    createdAt: fm.createdAt ?? new Date().toISOString(),
    updatedAt: new Date().toISOString(),
    lastVerifiedAt: new Date().toISOString(),
    lastUsedAt: fm.lastUsedAt,
    sourceRefs: uniqSourceRefs([...(fm.sourceRefs ?? []), ...candidate.sourceRefs]),
    tags: uniqStrings([...(fm.tags ?? []), ...candidate.tags]),
    aliases: uniqStrings([...(fm.aliases ?? []), ...(candidate.aliases ?? [])]),
    relatedIds: fm.relatedIds ?? [],
    supersedes: fm.supersedes ?? [],
    supersededBy: fm.supersededBy ?? [],
    sensitivity: fm.sensitivity ?? "normal",
    retention: fm.retention ?? "standard",
    archivePolicy: fm.archivePolicy ?? "auto",
    validation: candidate.validation ?? fm.validation
  };

  const body = [
    "## Summary",
    candidate.summary,
    "",
    "## Facts",
    ...candidate.facts.map(x => `- ${x}`),
    "",
    "## Evidence",
    ...candidate.sourceRefs.map(x => `- ${x.file}${x.anchor ? `#${x.anchor}` : ""}`),
    "",
    "## Open Questions",
    "",
    "## Revision History",
    `- ${new Date().toISOString()} upserted from candidate`
  ].join("\n");

  return matter.stringify(body, mergedFrontmatter as any);
}

function uniqStrings(values: string[]): string[] {
  return [...new Set(values.filter(Boolean))];
}

function uniqSourceRefs(refs: Array<{ file: string; anchor?: string }>): Array<{ file: string; anchor?: string }> {
  const map = new Map<string, { file: string; anchor?: string }>();
  for (const ref of refs) {
    map.set(`${ref.file}#${ref.anchor ?? ""}`, ref);
  }
  return [...map.values()];
}
```

```ts
import { DailyNoteEvent, MemoryCandidate } from "../types/memory";

export class CandidateExtractor {

          title: `Lesson ${event.project ?? "General"}`,
          summary: event.text,
          facts: [event.text],
          sourceRefs: [event.sourceRef],
          tags: ["incident", ...(event.tags ?? [])],
          scope: event.project ? "project" : "system"
        });
      }
    }

    return candidates;
  }
}
```

```ts
// src/memory/consolidation/consolidation-engine.ts

import { WorkspaceScanner } from "../scanner/workspace-scanner";
import { parseDailyEvents } from "../utils/frontmatter";
import { CandidateExtractor } from "./candidate-extractor";
import { MemoryScorer } from "../scoring/memory-scorer";
import { MarkdownStore } from "../store/markdown-store";
import { MemorySummaryWriter } from "../store/memory-summary-writer";

export class ConsolidationEngine {
  constructor(
    private scanner: WorkspaceScanner,
    private extractor: CandidateExtractor,
    private scorer: MemoryScorer,
    private store: MarkdownStore,
    private summaryWriter: MemorySummaryWriter
  ) {}

  async runDaily(): Promise<void> {
    const docs = await this.scanner.scanByKind("episodic");
    const todayDocs = docs.slice(-2); // 简化：取最近两天

    const candidates = [];
    for (const doc of todayDocs) {
      const events = parseDailyEvents(doc.content, doc.path);
      candidates.push(...this.extractor.extract(events));
    }

    for (const candidate of candidates) {
      const scored = this.scorer.score(candidate, {
        recallCount: 2,
        avgRecallQuality: 0.75,
        uniqueQueries: 2,
        daysObserved: 1,
        lastSeenAt: new Date().toISOString(),
        userConfirmed: false,
        validationSuccessCount: candidate.validation?.successCount ?? 0,
        validationFailureCount: candidate.validation?.failureCount ?? 0,
        conflictDetected: false,
        sensitivityDetected: false
      });

      if (scored.promotionEligible) {
        await this.store.upsertFromCandidate(candidate);
      }
    }

    await this.summaryWriter.refresh();
  }
}
```

## 11. MEMORY.md 汇总器骨架

因为 `MEMORY.md` 会占上下文，所以一定要可控生成。OpenClaw 官方对这一点讲得很明确，因此汇总器建议作为独立模块存在。[Source](https://docs.openclaw.ai/concepts/system-prompt)

```ts
// src/memory/store/memory-summary-writer.ts

import { WorkspaceScanner } from "../scanner/workspace-scanner";
import { MarkdownStore } from "./markdown-store";
import { MemoryConfig } from "../config/memory-config";

export class MemorySummaryWriter {
  constructor(
    private scanner: WorkspaceScanner,
    private store: MarkdownStore,
    private config: MemoryConfig
  ) {}

  async refresh(): Promise<void> {
    const docs = await this.scanner.scanAll();

    const semantic = docs
      .filter(d => d.layer === "semantic" && d.frontmatter?.status === "active")
      .sort((a, b) => (b.frontmatter?.confidence ?? 0) - (a.frontmatter?.confidence ?? 0));

    const procedural = docs
      .filter(d => d.layer === "procedural" && d.frontmatter?.status === "active")
      .sort((a, b) => (b.frontmatter?.confidence ?? 0) - (a.frontmatter?.confidence ?? 0));

    const preferences = semantic
      .filter(d => d.frontmatter?.type === "semantic.user")
      .slice(0, this.config.summary.maxPreferences);

    const projects = semantic
      .filter(d => d.frontmatter?.type === "semantic.project")
      .slice(0, this.config.summary.maxProjects);

    const procedures = procedural
      .slice(0, this.config.summary.maxProcedures);

    const markdown = [
      "# Core Identity",
      ...preferences.map(d => `- ${d.summary ?? d.title}`),
      "",
      "# Active Projects",
      ...projects.map(d => `- ${d.frontmatter?.title} -> ${d.path}`),
      "",
      "# Stable Preferences & Facts",
      ...semantic
        .filter(d => d.frontmatter?.type !== "semantic.project" && d.frontmatter?.type !== "semantic.user")
        .slice(0, this.config.summary.maxItems)
        .map(d => `- ${d.frontmatter?.title}: ${truncate(d.summary ?? "", 100)}`),
      "",
      "# Frequently Useful Procedures",
      ...procedures.map(d => `- ${d.frontmatter?.title} -> ${d.path}`),
      ""
    ].join("\n");

    await this.store.writeSummary(markdown);
  }
}

function truncate(text: string, n: number): string {
  return text.length <= n ? text : `${text.slice(0, n)}...`;
}
```

## 12. 总服务入口骨架

最后给一个外部最容易调用的 façade。

```ts
// src/memory/services/memory-service.ts

import { RecallQuery, RecallResult } from "../types/retrieval";
import { MemoryCandidate } from "../types/memory";
import { MemoryRetriever } from "../retrieval/memory-retriever";
import { MarkdownStore } from "../store/markdown-store";
import { ConsolidationEngine } from "../consolidation/consolidation-engine";
import { MemorySummaryWriter } from "../store/memory-summary-writer";
import { MemoryArchiver } from "../archive/memory-archiver";
import { WorkspaceScanner } from "../scanner/workspace-scanner";

export class MemoryService {
  constructor(
    private retriever: MemoryRetriever,
    private store: MarkdownStore,
    private consolidation: ConsolidationEngine,
    private summaryWriter: MemorySummaryWriter,
    private archiver: MemoryArchiver,
    private scanner: WorkspaceScanner
  ) {}

  async recall(query: RecallQuery): Promise<RecallResult> {
    return this.retriever.recall(query);
  }

  async upsertCandidate(candidate: MemoryCandidate): Promise<string> {
    const path = await this.store.upsertFromCandidate(candidate);
    await this.summaryWriter.refresh();
    return path;
  }

  async consolidateDaily(): Promise<void> {
    await this.consolidation.runDaily();
  }

  async runArchiveSweep(): Promise<void> {
    const docs = await this.scanner.scanAll();
    const decisions = docs
      .map(doc => this.archiver.decide(doc))
      .filter(Boolean);

    if (decisions.length > 0) {
      await this.archiver.execute(decisions as any);
      await this.summaryWriter.refresh();
    }
  }
}
```

## 14. 两个实现提醒

第一，**检索器不要自作主张把 `MEMORY.md` 变长**。因为官方机制下它会自动进入上下文，所以一定要把它当“摘要视图”，而不是全文知识库。[Source](https://docs.openclaw.ai/concepts/system-prompt)

第二，**评分器和归档器先规则化，再智能化**。OpenClaw Dreaming 的价值在于“晋升有门槛”，但工程上第一版没必要一开始就做复杂模型评分，先用可解释规则把主干跑通更稳。[Source](https://docs.openclaw.ai/concepts/dreaming)
